## 5.1. 제어 구조

## 5.1. if-else 문

```c
if (num > 0)
    printf("양수입니다");
```

괄호 안에 조건식을 넣고, 중괄호 안에 실행될 식을 넣는다. 

유의사항
1. 조건식은 반드시 괄호 안에 들어가야 한다
2. 실행될 식에 semicolon(;)을 찍어줘야 한다

if문 예시

```c
#include <stdio.h>
int main(void)
{
    int num;
    printf("정수를 입력하세요: ");
    scanf("%d", &num);

    if (num > 0)
    {   
        printf("양수입니다");
    }

    else 
    {
        printf("양수가 아닙니다");
    }
    
    return 0;
}
```
